## 📋 Descripción

Sitio web corporativo de Procomin, empresa colombiana especializada en la micropulverización y beneficio de minerales industriales no metálicos. La plataforma presenta nuestros productos principales: carbonato de calcio, caolín, talco y barita.

## 🌐 Sitio en Vivo

**URL:** [https://procomin.co](https://procomin.co)

## 🎯 Características

- ✨ Diseño moderno y responsivo
- 📱 Optimizado para dispositivos móviles
- 🎨 Mantiene la identidad visual de la marca
- 📧 Contacto directo simplificado (email y WhatsApp)
- 🖼️ Galería de productos con información detallada
- ⚡ Carga rápida y optimizada

## 🛠️ Tecnologías

## 🚀 Instalación y Uso Local

### Requisitos Previos
- Navegador web moderno
- Editor de código (VSCode recomendado)
- Servidor local (opcional: Live Server, XAMPP, etc.)

### Pasos de Instalación

1. Clonar el repositorio:
```bash
git clone https://github.com/tu-usuario/procomin.git
cd procomin
```

2. Abrir con Live Server o simplemente abrir `index.html` en tu navegador

## 🔧 Desarrollo

### Para modificar el sitio:

1. Edita el contenido en `index.html`
2. Ajusta estilos en `assets/css/styles.css`
3. Modifica funcionalidades en `assets/js/main.js`
4. Optimiza imágenes antes de subirlas a `assets/img/`

### Buenas Prácticas:
- ✅ Optimizar imágenes (WebP cuando sea posible)
- ✅ Mantener código limpio y comentado
- ✅ Testear en múltiples dispositivos
- ✅ Validar accesibilidad (WCAG AA)
- ✅ Verificar SEO antes de deploy
